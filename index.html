ghrr: GitHub Hosted R Repository

 

About

This ghrr (for "GitHub Hosted R Repository") uses drat for both insertion of packages, and usage from R.

Setup

Simply install the drat CRAN package> the usual way via install.packages("drat", repos="http://cran.studio.com").

Then add one line to your .Rprofile file (or its site-wide equivalent:
drat:::add("ghrr")

That's it! Now use install.packages() or update.packages() and the ghrr will be consider just like the standard repositories for R.

More

You can learn more about drat from the vignettes Drat for Package Users and Drat for Package Authors.
