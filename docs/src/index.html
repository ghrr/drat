ghrr drat

This GitHub Hosted R Repository ("ghrr") repository uses drat for both insertion of packages, and usage from R.

Setup Method One: Simply install the drat CRAN package the usual way via install.packages("drat", repos="https://cloud.r-project.org") and then add one line to your .Rprofile file (or to Rprofile.site): drat:::add("ghrr"). That's it! Now use install.packages() or update.packages() and this ghrr drat will be considered along with the standard repositories for R.

Setup Method Two: A direct approach can pass the repo information to install.packages() (and to update.packages()): install.packages("AnomalyDetection", repo=c("https://ghrr.github.io/drat", "https://cloud.r-project.org")) (which also passes the CRAN repo information in case other depencies are needed, add BioConductor or other repos as needed too).

You can learn more about drat at the package documentation site which includes several vignettes.
